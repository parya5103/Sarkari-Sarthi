# SarkariSarthi 2.0 🚀

**Smart Indian Sarkari Job Portal**

[![Python 3.11+](https://img.shields.io/badge/python-3.11+-blue.svg)](https://www.python.org/downloads/)
[![Telegram](https://img.shields.io/badge/Telegram-Join%20Channel-blue.svg)](https://t.me/sarkarisarthi)

SarkariSarthi 2.0 is your destination for Indian government job seekers. It brings you the latest sarkari job updates and provides instant notifications through Telegram.

## ✨ Features

### 🔍 Advanced Job Searching
- **Multi-Source Updates**: Get updates from popular Indian job sites.
- **Smart Filtering**: Find jobs easily by categories like Banking, SSC, Railway, Police, Defence, Teaching, UPSC, etc.
- **Instant Details**: All the important dates, requirements, and application links in one place.

### 📱 Modern Interface
- **Glassmorphism Design**: Beautiful, modern UI with glass-like effects.
- **Dark Mode Support**: Toggle between light and dark themes.
- **Responsive Design**: Works perfectly on desktop, tablet, and mobile.

### 🔔 Instant Notifications
- **Telegram Integration**: Real-time job alerts sent to our Telegram channel.

## 🚀 Quick Start

### Prerequisites
- Modern Web Browser
- Internet Connection

### Usage

1. Open the website.
2. Search for jobs using the search bar or filter by category.
3. Click on any job card to view detailed information.
4. Click "Apply Now" to go to the official application page.
