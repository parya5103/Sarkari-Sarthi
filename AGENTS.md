# Jules Agent Configuration

## Tech Stack
* Framework: Next.js (App Router), React, TypeScript
* Styling: TailwindCSS
* Deployment: Vercel / Firebase

## Coding Standards
* Write clean, modular, and mobile-first responsive components.
* Prioritize robust error handling and loading states.
* Resolve technical errors independently and provide the final working code directly. Do not explain every error step-by-step.
* Ensure all APIs are secure and environment variables are strictly managed.
